Правила формирования заказов

  • Заказ должен быть сохранен не позднее 17:00 накануне дня доставки.
  • Отдельные блюда заказываются по цене, указанной в меню.
  • Можно заказать комплексный обед (комплекс) с фиксированной стоимостью.
  • В «комплекс» объединяются: суп, салат и второе блюдо с гарниром, либо самостоятельное блюдо. Суп можно поменять на дополнительный салат и наборот.
  • В одном заказе может быть только один комплекс, все остальные блюда рассчитываются по их цене из меню.
  • Если блюд выбрано больше, чем на один комплекс, они объединяются из расчета максимальной цены.
  • Если меню будет изменено руководством кафе, к вам на почту придёт уведомление.
